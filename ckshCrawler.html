Cksh Crawler

Cksh Crawler is a simple tool that can sort the latest information from CKSH's official website by time order. The latest news on the official CKSH website isn't sorted by date. Instead, they are sorted by Importance. This method may be useful for those who aim to make acquaintance with the school. However, it's extremely inconvenient for students that want to receive the latest news. So I coded this little project with Python, and utilize Python's module "Tkinter" for the UI.
